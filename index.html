đối với text thì chúng ta có thể thêm màu bằng thuộc tính color { color: red; }


điều chỉnh vị trí của text bằng text-align


chữ này canh trái { text-align: left; }

chữ này canh giữa { text-align: center; }

chữ này canh phải { text-align: right; }



có thể trang trí cho text bằng text-decoration { text-decoration: underline; }


chỉnh thành viết hoa bằng text-transform { text-transform: uppercase; }


letter spacing điều chỉnh khoảng cách của chữ { letter-spacing: 5px; }


line heigh điều chỉnh khoảng cách giữa các dòng

dòng này đc chỉnh gần hơn { line-height: 0.1; }
